# 🍽️ RecipeHub - Recipe Sharing Platform

## 📌 Project Overview

RecipeHub is a full-stack recipe sharing platform where users can discover, create, and manage delicious recipes from different cuisines and categories. The platform offers secure authentication, premium membership, recipe purchasing, favorites, likes, reporting, and a complete admin dashboard for managing users, recipes, reports, featured recipes, and transactions.

## ✨ Key Features

- 🔐 Secure authentication with Email/Password and Google Sign-In using Better Auth
- 👤 User profile management with premium membership support
- 🍽️ Create, update, and delete personal recipes
- 🔍 Browse, search, and filter recipes by category
- ❤️ Like and unlike recipes (one like per user)
- ⭐ Add and remove favorite recipes
- 🚩 Report inappropriate recipes to administrators
- 💳 Purchase premium recipes securely using Stripe
- 👑 Upgrade to Premium Membership through Stripe Checkout
- 📖 View purchased premium recipes anytime
- 🌟 Featured recipes and popular recipes sections on the homepage
- 📱 Fully responsive design for mobile, tablet, and desktop
- 🛠️ Powerful Admin Dashboard with analytics and management features
